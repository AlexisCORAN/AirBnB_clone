# AirBnB clone - The console

## Description
In this project we have defined all the models of classes and functions needed to make a basic clone of AirBnB

## Classes

| BaseModel  | Defines the common structure other classes |
|    ---     |                    ---                     |
| Amenity    | Defines a class for Amenity                |
| City       | Defines a class for City                   |
| Place      | Defines a class for Place                  |
| Review     | Defines a class for Review                 |
| State      | Defines a class for State                  |
| User       | Defines a class for User                   |
| FileStorage | Defines a class that will handle the serialization/deserialization data process in JSON format |
| HBNBCommand | Defines a class for a console that inherits from `cmd.Cmd` |

## Console
When the file `console.py` is executed, it act as a console on which we can execute commands to manipulate data.
*Usage*: `$ ./console.py`

### Allowed commands and syntax
- `quit`, `EOF` : Exists the console terminating the program
*Usage:* `$ quit`, `$ EOF`

- `help`: Displays information about the allowed commands
*Usage:* `$ help`

- `create`: Creates a new instance of a specified class, saves it to a JSON file and prints the `id` of the newly created instance
*Usage:* `$ create <class name>`

- `show`: Prints the string representation of an instance based on the class name and `id`
*Usage*: `$ show <class name> <id>` or `$ <class name>.show(<id>)`

- `destroy`: Deletes an instance based on the class name and id and saves the change into the JSON file
*Usage*: `$ destroy <class name> <id>` or `$ <class name>.destroy(<id>)`

- `all`:  Prints all string representation of all instances based or not on the class name, if no classname was specified then it prints all objects of every class
*Usage*: `$ all`, or `$ all <class name>` or `$ <class name>.all()`

- `update`: Updates an instance based on the class name and id by adding or updating attribute and saves the change into the JSON file
*Usage*: `$ update <class name> <id> <attribute name> "<attribute value>"` or `$ <class name>.update(<id>, <attribute name>, <attribute value>)` or `$ <class name>.update(<id>, <dictionary representation>)`

- `count`: Print the number of instances of a specfied class
*Usage*: `$ <class name>.count()`
